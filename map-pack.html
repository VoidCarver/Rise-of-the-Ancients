Currently Live - Version 2.2

General Changes

  • 2 Player Size changed from 120x120 to 144x144 (3 Player Size)
    (intended in allowing to match number of players vs map size on a larger 2P size than official maps)
  • Lurable animals (Elephants/Boars) now spawn closer to players (around 20 tiles).
  • Maps without specific changes noted below only use the 2 changes noted above.
  • Mega Random is entirely untouched, it was added to ease selection and avoid swapping to default maps.

Citadel

  • Base and surrounding forest generation changed to match the map size change.
  • There should no longer be unconnected walls/forest on the front base.
  • Back of every base should now have lot more space to build.

Coastal

  • Chance for map generation are now :
    1 Island: 80% (previously 60%)
    2 Islands: 20% (previously 40%)

Inside Out

  • Deers now always spawn inside the walls and no longer outside in the back.
    (Originally 50/50% chance to spawn either inside or outside in the back).

Muddy Waters

  • Each players now starts with 2 Natural Elephants/Boars instead of 1.

Oasis

  • Changed secondary Forests size from 7 to 9 (land %)

Plateaus

  • Adjusted Player Land with 2 Players on new 2 Players map size.
    (Player Land were way too small with 2 Players on a 3 Players map like size resulting in Players being enclosed by wood and rest of the desert region being a no man's land due to missing a player for generation).
  • Added additional small patches of forest (1 land% size) within the desert region on 2 Players size.
  • The map should now feel again a lot more like the original 2 Players on default 2 Players Map Size.
  • Fixed a bug from Official Map where you could "capture" neutral Gold/Stone mines.

Sahara

  • Changed secondary Forests size from 3 to 5 (land %)
  • You can find below the changelog of Community Map Pack changes.
